## Part 1: Parameter Recovery

### Method

- **Simulation design:** n = 30 synthetic subjects × 600 trials × 50 independent repetitions
- **Ground truth:** Ω ~ Uniform(0.5, 3.0), σ ~ Uniform(0.05, 0.45)
- **Load sequence:** E linearly increasing from −0.5 to 2.0 (matching planned protocol)
- **Fitting method:** Analytical OLS rearrangement of the Euler–Maruyama equation

The OLS rearrangement is exact in the small-dt limit. Rearranging the
discretised Langevin equation:

```
Δβ/dt + β³ − E = Ω·β + noise
```

yields Ω̂ = (β' y) / (β' β) where y = Δβ/dt + β³ − E.
σ̂ is estimated from the standard deviation of residuals rescaled by √dt.

## Part 2: Model Discrimination

### Method

- **Models compared:** TGC (cusp Langevin), DDM/OU (linear), HMM (2-state),
  Hopf (Stuart-Landau), Neural-field (saturating linear)
- **Classification criterion:** BIC — lowest BIC model is selected
- **Design:** 60 synthetic datasets per model (300 total), each with 600 trials
- **Fitting:** Analytical OLS for continuous models; grid-search forward algorithm for HMM

BIC = 2 × NLL + k × ln(n), where k is the number of free parameters
(k = 2 for TGC, DDM/OU, Hopf; k = 3 for Neural-field; k = 5 for HMM).

## Discriminating Features Summary

The five models produce qualitatively different trajectories in three dimensions:

| Model | Bistable | Oscillatory | Load-dependent threshold |
|---|---|---|---|
| TGC (Cusp)      | Yes (Ω > 0) | No  | Yes |
| DDM / OU        | No          | No  | Yes (linear) |
| HMM (2-State)   | Effectively | No  | No (discrete) |
| Hopf            | No          | Yes | Partial |
| Neural-field    | Weak        | No  | Yes (saturating) |

The TGC model uniquely combines bistability, load-dependent threshold,
and noise-induced transitions. See `simulation/model_comparison.py` for
a figure illustrating these differences across trajectory, distribution,
potential landscape, and autocorrelation panels.
